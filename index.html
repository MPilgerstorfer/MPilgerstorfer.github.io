Project Objectives

  • Enable the drone to navigate autonomously using the Jetson Nano’s processing power.
  • Utilize the Kinect sensor for real-time object detection and tracking.
  • Collect and analyze data from drone flights for various applications such as environmental monitoring and 3D mapping.
  • Develop an intuitive user interface for monitoring and controlling the drone.

Hardware Components

DJI Tello Drone

The DJI Tello is a small, lightweight drone known for its ease of use and versatility.

Jetson Nano

The Jetson Nano is a compact AI computer that provides powerful computing capabilities.

Kinect Sensor

The Kinect sensor adds advanced depth-sensing and object recognition capabilities.

Project Phases

Phase 1: Initial Setup and Configuration

Assemble hardware, install software, and conduct basic flight tests.

Phase 2: Autonomous Navigation

Calibrate and integrate the Kinect sensor, develop path planning algorithms, and conduct tests.

Phase 3: Object Detection and Tracking

Collect training data, train AI models, and implement real-time object tracking.

Phase 4: User Interface Development

Create a user-friendly interface and develop tools for visualizing data.

Phase 5: Data Analysis and Application

Analyze flight data, create 3D maps, and generate reports.
